# Loan Prediction System

A beginner Machine Learning project that predicts whether a loan will be approved or rejected using a Support Vector Machine (SVM) classifier.

This project was built as part of my Machine Learning learning journey and helped me understand the complete workflow of a supervised classification problem.

---

## Concepts Used

* Data Preprocessing  
* Handling Missing Values  
* Label Encoding  
* Feature Selection (X and Y separation)  
* Train-Test Split  
* Support Vector Machine (SVM)  
* Model Evaluation (Accuracy Score)  

---

## Results

* Training Accuracy: 79.8%
* Testing Accuracy:  83.3%

---

## What I Learned

Through this project, I learned how to clean and prepare real-world data, convert categorical values into numerical form using encoding techniques, train an SVM model using scikit-learn, evaluate its performance, and build a simple prediction system that works on new input data.

I also understood the importance of preprocessing, especially how models only accept numerical inputs and how feature selection affects model performance.
